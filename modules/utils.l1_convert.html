L1 Convert Module#

Functions for converting L1C with DN Rrs to decimal and with angles needed for ACOLITE.

Level-1 to reflectance conversion utilities.

Provides functions to convert Sentinel-2 L1C Digital Numbers to top-of-atmosphere (TOA) reflectance with proper geometric metadata.

gee_acolite.utils.l1_convert.DN_to_rrs(image: Image) → Image[source]#

Convert L1C image from Digital Numbers to TOA reflectance.

Applies radiometric calibration (DN / 10000) and extracts viewing geometry from metadata (solar and viewing zenith/azimuth angles). Also computes relative azimuth angle.

Parameters:

image (ee.Image) – Sentinel-2 L1C image with DN values and angle metadata.

Returns:

Image with TOA reflectance (0-1) and geometry properties: ‘sza’, ‘saa’, ‘vza’, ‘vaa’, ‘raa’.

Return type:

ee.Image

gee_acolite.utils.l1_convert.get_mean_band_angle(image: Image, angle_name: str) → Number[source]#

Compute mean viewing angle across all bands.

Extracts and averages band-specific viewing angles from Sentinel-2 metadata. Each band has its own viewing geometry due to the sensor’s detector layout.

Parameters:

  • image (ee.Image) – Sentinel-2 L1C image with angle metadata.

  • angle_name (str) – Angle type to extract: ‘ZENITH’ or ‘AZIMUTH’.

Returns:

Mean viewing angle across all bands (in degrees).

Return type:

ee.Number

gee_acolite.utils.l1_convert.l1_to_rrs(images: ImageCollection, scale: int) → ImageCollection[source]#

Convert L1C image collection from DN to TOA reflectance.

Applies DN to reflectance conversion and resamples all bands to the specified spatial resolution.

Parameters:

  • images (ee.ImageCollection) – Sentinel-2 L1C image collection with DN values.

  • scale (int) – Target spatial resolution in meters (10, 20, or 60).

Returns:

Collection with TOA reflectance and geometry metadata.

Return type:

ee.ImageCollection

gee_acolite.utils.l1_convert.resample(image: Image, band: str) → Image[source]#

Resample all bands to match a reference band’s projection.

Applies bilinear resampling to align all bands to the spatial resolution and projection of the specified reference band.

Parameters:

  • image (ee.Image) – Image with multiple bands at different resolutions.

  • band (str) – Reference band name (e.g., ‘B2’ for 10m, ‘B5’ for 20m).

Returns:

Resampled image with all bands matching reference projection.

Return type:

ee.Image
