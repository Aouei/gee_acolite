Masks Module#

Image masking functions. Functions for creating and applying masks to imagery.

Masking utilities for quality control and scene filtering.

Provides functions to create various masks for cloud detection, water/land classification, and quality filtering.

gee_acolite.utils.masks.add_cld_shdw_mask(img: ee.Image, cloud_prob_threshold: float = 50, nir_dark_threshold: float = 0.15, cloud_proj_distance: float = 1, buffer: int = 50) → ee.Image[source]#

Add combined cloud and shadow mask with buffer.

Creates comprehensive cloud and shadow mask by combining: 1. Cloud probability mask 2. Cloud shadow detection 3. Buffer around masked areas

Parameters:

  • img (ee.Image) – Image with ‘cloud_prob’ property containing Cloud Probability data.

  • cloud_prob_threshold (float, optional) – Cloud probability threshold 0-100 (default: 50).

  • nir_dark_threshold (float, optional) – Threshold for dark NIR pixels 0-1 (default: 0.15).

  • cloud_proj_distance (float, optional) – Shadow projection distance in km (default: 1).

  • buffer (int, optional) – Buffer distance around clouds/shadows in meters (default: 50).

Returns:

Input image with ‘cloudmask’ band (1 = cloud/shadow, 0 = clear).

Return type:

ee.Image

gee_acolite.utils.masks.add_shadow_bands(img: ee.Image, nir_dark_threshold: float = 0.15, cloud_proj_distance: float = 1) → ee.Image[source]#

Add cloud shadow mask bands to L1C image.

Identifies cloud shadows by projecting cloud locations in the direction of solar illumination and intersecting with dark NIR pixels.

Parameters:

  • img (ee.Image) – Image with ‘clouds’ band and solar geometry metadata.

  • nir_dark_threshold (float, optional) – Threshold for dark pixels in NIR (0-1, default: 0.15).

  • cloud_proj_distance (float, optional) – Maximum shadow projection distance in km (default: 1).

Returns:

Input image with ‘dark_pixels’, ‘cloud_transform’, and ‘shadows’ bands.

Return type:

ee.Image

gee_acolite.utils.masks.non_water(image: ee.Image, band: str = 'rhot_B11', threshold: float = 0.05) → ee.Image[source]#

Create water/land classification mask.

Uses SWIR reflectance to distinguish water from land. Water has low SWIR reflectance due to strong absorption.

Parameters:

  • image (ee.Image) – Image with SWIR band.

  • band (str, optional) – SWIR band name (default: ‘rhot_B11’).

  • threshold (float, optional) – Maximum reflectance for water (default: 0.05).

Returns:

Binary mask (1 = water, 0 = land).

Return type:

ee.Image
